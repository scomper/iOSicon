# iOSicon

iOS 14 上利用捷径可以自定义 App 的图标。
- 安装 [Icon Themer](https://routinehub.co/shortcut/6565/)捷径
- 运行捷径后选择自己的 icon 图标文件。

使用方法可以阅读[「iOS 14 上替换应用图标」](https://scomper.me/ios/2020-10-17)
